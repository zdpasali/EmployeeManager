# EmployeeManager

Full-stack employee management system built with Node.js, Express, MongoDB, and Angular.

EmployeeManager is a web application that allows managing employee records within an organization. The application provides functionality for listing, creating, and updating employees through a RESTful API and Angular frontend.

## Features

- Display employee list from MongoDB
- Create new employee with form validation
- View detailed employee information
- Update employee records
- Profile image support
- Structured backend (controllers, routes, models)
- REST API communication between frontend and backend

## How It Works

On application start, users can click “Show Employees” to retrieve and display records stored in the database.

New employees can be created through a validated form.  
Each employee has a detailed view component where additional information can be reviewed and updated. Required fields must be completed before submission.

## Tech Stack

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

Frontend:
- Angular
- Angular Material

## Purpose

This project was developed as a technical recruitment assignment to demonstrate full-stack development skills, including REST API design, database modeling, frontend integration, and form validation.
